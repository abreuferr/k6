# Realizar login na cloud

### Login com e-mail e senha

```
$ k6 login cloud
```

#### Preencha:
 - Email:
 - Senha:

### Login com token

```
$ k6 login cloud --token 5646gh684h64hgfd
```

#### Para obter o token
1. Vá até a plataforma do k6 cloud.
2. Acesse "Settings".
3. Clique em "Personal API Token".
4. Copie o token.
